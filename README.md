# DreamCastle

DreamCastle is a career roadmap platform that turns a job goal or vacancy into a gamified path of skill gates, readiness checks, progress and preparation tasks.

## Features

- Landing page
- Create roadmap form
- Dynamic gate roadmap
- Gate checks
- LocalStorage progress
- React Router navigation

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Motion
- Lucide React
